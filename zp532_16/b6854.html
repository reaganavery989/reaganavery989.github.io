LEATHERMAN WAVE / Discontinued model

■ About　LEATHERMAN WAVE



LEATHERMAN 社を代表するツールであり、1998年にファーストモデルの登場から25年の歳月が経つロングセラーモデルでプライヤー型マルチツールでは世界一の累計販売数を誇るモデルが WAVE です。



とても人気の高いモデルで現行モデルは WAVE＋で WAVE は Discontinued model となって市場には流通していません。



現行モデルの WAVE＋との差異は画像２の赤色〇で示したプライヤーの基部にあるワイヤーカッターが交換可能な脱着式になった以外にマイナーチェンジした箇所がないことから25年前に登場した時点で改善・改良する余地がない完成度を誇るマルチツールと言っても過言ではないモデルです。



そもそも連日の作業で頻繁にワイヤーを切断する必要がある職種に就いていらっしゃる御方以外にワイヤーカッターが刃毀れすることは僅少だと思います。



■ 25年保証を考察



私事で恐縮ですが４ストロークに比べて構造がシンプルな２ストロークエンジンであれば O / H ( 分解・修理交換・組立・調整 ）できるスキルがあるため、ラチェット・レンチなら Snap-on 、KTC 、プライヤー系だと KNIPEX と拘って買い揃えたハンドツール群を数多く所有していますが、これまでハンドツール自体が作業中に破損・欠損・ピボットが歪む等で使用できなくなった例など身に覚えがありません。



ここからは当方の主観ですが、世間で一級品と称賛されるハンドツールメーカーの製品であれば、酷使しても破損することは皆無に等しいため LEATHERMAN 正規輸入代理店レザーマンツールジャパン（株）が謳う『 日本国内での25年保証が付帯 』をユーザーが実際に利用する事態に直面することは僅少で保証が付帯しているという安心感をユーザーの心理に訴求することで WAVE＋の税込価格￥22,000- は相対的な損得とはあまり関係がない、又は損得がぼかされてしまう心理的価格の『 名声価格 』を設定することで商品を意図的に高く販売することが可能になるビジネス戦略だと感じています。
