【商品説明】
焚き火や調理を行うキャンプシーンに適したエプロン。難燃性が高く、耐久性に優れたコーデュラナイロン生地を全体に使用しています。生地に炎が着くと微量のガスが燃焼部分の酸素を奪い、無酸素状態にすることで消火。さらに火がついた部分はすぐに炭化するため、溶けた生地の液垂れによる火傷のリスクも低減します。ウエスト部分にグローブやハンマーを収納できる大容量のポケットと、左胸には貴重品や小物を収納できるジッパーポケットを配置。両肩はストラップでサイズ調節ができ、着脱はマグネットで簡単にできます。年代を問わず着用しやすい、ベーシックなデザインです。
■カラー：AN ( アビエイターネイビー )
■サイズ：F ( 胸幅/27cm、ウエスト/79cm、裾囲/79cm、前丈/89cm、脇丈/61cm )
■素材：Fire Resistant Nylon Cloth ( アクリル系75%、ナイロン25% )
■機能： 着炎すると放たれる極微量のガスにより、燃焼部分周辺を瞬時に無酸素状態にして自己消化の働きを促す特殊繊維素材です。着炎部を炭化することで繊維の溶融を防ぎ、溶融物による火傷のリスクを低減させます。 一般的なナイロンの約7倍もの強度を持つと言われる高耐久性の特殊素材。非常に優れた頑丈さにくわえ、素材が持つ光沢感や手触りの良さも特長。
